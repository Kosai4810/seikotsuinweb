# 整骨院web 事業・営業・サイト改善メモ

最終更新: 2026-07-12

## 事業コンセプト

整骨院・接骨院・鍼灸院・整体院に特化して、ホームページ制作だけでなく、患者さんが院を見つけて、比較して、問い合わせるまでのWeb導線を整える。

単なる「低価格ホームページ制作」ではなく、「治療院のWeb導線スターターパック」として訴求する。

## サイトの主な訴求

- ホームページ制作から、集客導線づくりまで対応する。
- 5ページ程度、スマホ対応、問い合わせフォーム、電話・メール導線、Googleマップ導線、基本SEO設定、公開後1か月サポートをモニター制作プランに含める。
- 通常128,000円（税込）〜のところ、先着3院限定で98,000円（税込）〜。
- 既存HPがある院には「無料HP診断」として改善点を3つ整理する。
- HPがない院には、開業・集客に必要な導線を整理する。
- 制作担当が直接ヒアリングする。
- 小規模だからこそ、柔軟・低価格・丁寧に対応できる。

## 98,000円で提供できる理由

- 治療院向けに必要なサイト構成をあらかじめ整理しているため。
- モニター掲載、制作事例掲載、感想掲載に協力してもらう前提のため。
- 最初から大規模に作り込まず、必要なものから始めて、反応を見ながら拡張する設計のため。

## 提供サービス

- ホームページ制作
- 予約・問い合わせ導線設計
- Googleビジネスプロフィール、Googleマップ導線整備
- 公式LINE・予約導線設計
- Instagram初期設計
- Google広告などの広告運用
- アクセス流入管理・改善提案
- 開業前のWeb導線設計
- チラシ・名刺・簡易ビジュアル制作
- 公開後の保守・運用サポート

## 営業リスト分類

営業対象は、各院のHP状態でA〜Cに分類する。

- A: クオリティの高いHPが存在する
- B: HPはあるがクオリティが低い、情報が古い、導線が弱い
- C: そもそもHPがない

## 営業優先度

- ティア1: 開業して半年未満のC。最優先。
- ティア2: 開業半年以上のC。
- ティア3: 大きく改善点が見られるB。
- ティア4: 多少改善点が見られるB。
- ティア5: A。

## 営業方針

- 一斉送信ではなく、対象を絞った個別営業にする。
- 各院の情報を詳細に確認した上で、オリジナルの提案文を作る。
- Aの場合は、HP制作ではなく、不足している導線や運用面を提案する。
- Bの場合は、各院のHPを詳細に確認し、具体的な改善点をまとめて提案する。
- Cの場合は、開業・集客に必要な最低限のWeb導線を98,000円から整えられることを提案する。
- 返信なしの追客は1回まで。
- 拒否された院は送信禁止リストへ入れる。
- 営業メールの下書き、分類、追客状況、送信禁止情報はExcelで管理する。

## サイト改善の方向性

- 上品で落ち着いたブランドイメージ、配色、写真の方向性は維持する。
- ただし、詩的な表現だけではなく、院長が3秒で価値を理解できる直球の訴求を増やす。
- 「綺麗に作れそう」だけでなく、「集客導線まで考えてくれそう」と感じてもらう。
- 実績が増えるまでは、制作実績の代わりに診断サンプル・改善提案サンプルで提案力を見せる。
- 将来的には、実名または匿名の制作事例、お客様の声、問い合わせ増加などの数字を追加する。
