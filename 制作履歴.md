# 建設WEB 制作履歴

## 事業ブランド

- 親ブランド名: CraftLead
- 業種特化サイト名: 建設WEB
- 目的: 建設業者、職人、一人親方、小規模工務店向けに、低価格で問い合わせにつながるホームページ制作サービスを訴求する。

## 制作したサイト

- 技術: Next.js / React / TypeScript / Tailwind CSS
- ページ:
  - トップ
  - 料金
  - 対応業種
  - 制作の流れ
  - よくある質問
  - お問い合わせ
  - 特定商取引法に基づく表記
  - プライバシーポリシー

## 主な改善履歴

1. 建設業専門の格安ホームページ制作サイトとして初期構築。
2. 「なぜ低価格なのか？」セクションを追加。
   - 理由1: 建設業に特化しているから
   - 理由2: 問い合わせに必要な内容へ絞るから
3. 黒と黄色の強い配色から、白背景中心の清潔感ある配色へ調整。
4. 建設中・施工中のリアル寄り写真を追加。
5. 写真配置を、素材の横並びから「大きな施工写真 + 補助写真2枚」の構成へ変更。
6. ファーストビューのタイトルを画面横幅を活かす大きな見出しに調整。
7. 公式ロゴ画像をヘッダーとフッターへ反映。
8. 大手建設会社サイトを参考に、基本フォントを明朝体へ変更。
9. 設計図を連想させる薄いグリッド、細い罫線、角だけを強調するフレームを追加。
10. 見出し構成を「大見出し → 小ラベル」の順番に統一。
11. ファーストビュー背景を、薄い水色の青空を見上げる建設業者の写真へ変更。
12. 黒塗りのセクションを白背景へ変更し、列挙系コンテンツをコンパクトな多段表示へ整理。
13. `ほうかんWEB` のSEO構成を参考に、LocalBusiness / Service / FAQPage の構造化データを追加。
14. `robots.txt` と `sitemap.xml` を Next.js の MetadataRoute で生成。
15. 業種別SEOページと地域別SEOページを追加。
16. 集客コラム一覧ページとトップのSEO・ページ戦略セクションを追加。
17. 問い合わせ電話番号を `070-7615-5231` に統一。
18. `ほうかんWEB` の見せ方を参考に、トップの流動性とデザイン性を強化。
   - ファーストビューに施工写真が縦に流れる写真レールを追加。
   - 見出し、CTA、カードにフェードイン・浮き上がり・価格強調の演出を追加。
   - CTAボタンを丸型・影付きに変更し、問い合わせ導線を目立たせた。
   - 写真ブロックに斜めの切り抜きと影を加え、現場感を強めた。
   - ESLintがVercel生成物を拾わないよう `.vercel/**` を除外。
19. `ほうかんWEB` にさらに寄せ、形式・字体・余白感を統一。
   - 基本フォントを明朝体から `Noto Sans JP` ベースのゴシック体へ変更。
   - ファーストビューを白背景の左右分割構成へ変更。
   - スマホでも流れる写真レールが見える構成へ変更。
   - 丸型CTA、柔らかいカード、緑アクセントを中心に整理。
